Samenvatting toegankelijkheid http://127.0.0.1:5501/index.html

De website http://127.0.0.1:5501/index.html heeft enkele toegankelijkheidsproblemen die verbeterd kunnen worden:

**Kleurcontrast Het contrast tussen de achtergrond- en tekstkleuren is voldoende. → geen veranderingen nodig.

**Aanraakdoelen De klikbare elementen zijn te klein en staan te dicht op elkaar. → Oplossing: meer ruimte toevoegen tussen de aanraakdoelen (bijvoorbeeld met extra padding).

<img width="640" height="787" alt="image" src="https://github.com/user-attachments/assets/54c5e915-3e6a-498e-8dd2-2061c58e90f6" />

**Formulierelementen hebben geen duidelijke naam → Oplossing: geef de formulierelementen een duidelijkere naam
<img width="713" height="355" alt="image" src="https://github.com/user-attachments/assets/b5862cb3-c167-47d0-bb4f-54c221fff0dc" />

**Kopelementen (headings) De koppen staan in logische, aflopende volgorde. → geen veranderingen nodig.

**Afbeeldingen hebben duidelijke alt-teksten. → Oplossing: geen verranderingen nodig.


**Handmatige toegankelijkheidstests

*Keyboard-navigatie: werkt goed; bij het tabben worden knoppen netjes doorlopen.

*Screenreader-test: leest de tekst correct, maar afbeeldingen worden goed beschreven.

*Interactieve elementen: niet alle interactieve onderdelen hebben een hover-effect, maar het is wel duidelijk welke elementen klikbaar zijn doordat de cursor verandert.

*Landmarks: de site gebruikt correcte main- en nav-elementen met impliciete landmark-rollen.

**Kortom: de website is grotendeels functioneel, de structurele toegankelijkheid is top, vooral op het gebied van kleurcontrast, alt-teksten en headingvolgorde.
